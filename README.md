# Markdown Messenger Converter

Uma aplicação HTML premium de arquivo único para converter Markdown nos formatos do WhatsApp e do Google Chat.

## Funcionalidades
- **Formatação para WhatsApp**: Converte negrito, itálico e listas para a sintaxe compatível com o WhatsApp.
- **Formatação para Google Chat**: Otimizado para o suporte específico de Markdown do Google Chat.
- **Conversão em Tempo Real**: Veja suas alterações instantaneamente enquanto digita.
- **Interface Premium**: Interface moderna em modo escuro com glassmorphism elegante e animações.
- **Copiar para Área de Transferência**: Botões de acesso rápido para copiar o texto formatado.

## Como Usar
1. Abra o `index.html` em qualquer navegador moderno.
2. Digite ou cole seu Markdown na área de entrada.
3. Clique nos botões de copiar para o formato do WhatsApp ou do Google Chat.
